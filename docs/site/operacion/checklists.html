Índice Inicio Swagger

Descripción

Checklists por puesto para control operativo.

Rutas y permisos

MétodoRutaDescripciónQueryAuthRoles
POST/checklistsCrea checklist-SíADMIN, GERENTE
GET/checklistsLista checklistspage:number, pageSize:number, puestoId:number, nombre:string, meta:booleanSí-
GET/checklists/{id}Obtiene checklists-Sí-
PATCH/checklists/{id}Actualiza checklists-SíADMIN, GERENTE
DELETE/checklists/{id}Elimina checklists-SíADMIN, GERENTE

Controller (métodos)

  • list
  • create

Modelo (campos clave)

  • id
  • nombre
  • puestoId

Validaciones

  • puestoId válido
  • Paginación: page, pageSize
  • Filtros: puestoId, nombre
  • No se puede eliminar si tiene checklist_items relacionados.

Ejemplos

Request
{
  "nombre": "Entrega"
}
Response
{
  "id": 1,
  "nombre": "Entrega"
}

Errores comunes

  • 404 puesto no encontrado
  • 409 relación inválida
