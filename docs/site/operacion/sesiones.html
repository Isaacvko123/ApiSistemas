Sesiones

Control de refresh tokens

Índice Inicio Swagger

Descripción

Registra sesiones por usuario y permite revocación por sesión o global.

Rutas y permisos

MétodoRutaDescripciónAuthRoles
POST/auth/loginCrea sesión y tokensNo-
POST/auth/refreshRota refreshNo-
POST/auth/logoutRevoca sesión actualSí-
POST/auth/logout-allRevoca todas las sesionesSí-

Controller (métodos)

  • login
  • refresh
  • logout
  • logoutAll

Modelo (campos clave)

  • id
  • usuarioId
  • refreshHash
  • expiresAt
  • lastUsedAt

Validaciones

  • JWT válido
  • refreshHash coincide

Ejemplos

Request
{
  "email": "ana@empresa.com",
  "contrasena": "PasswordSegura123"
}
Response
{
  "accessToken": "...",
  "refreshToken": "..."
}

Errores comunes

  • 401 token inválido
  • 401 refresh revocado

Checklist operativo

  • Rotación refresh
  • Revocación global
  • Audit logs de login
