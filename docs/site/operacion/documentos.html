Descripción

Carga y registro de PDF/imagen relacionados a resguardos.

Rutas y permisos

MétodoRutaDescripciónQueryAuthRoles

GET/documentosLista documentospage:number, pageSize:number, equipoId:number, resguardoId:number, resguardoEquipoId:number, tipo:string, empleadoId:number, evento:stringSí-

POST/documentosCrea documentos-SíADMIN, GERENTE

Controller (métodos)

  • list
  • upload

Modelo (campos clave)

  • id
  • ruta
  • tipo
  • resguardoId
  • empleadoId

Validaciones

  • MIME permitido
  • tamaño <= límite
  • Paginación: page, pageSize
  • Filtros: equipoId, resguardoId, resguardoEquipoId, tipo, empleadoId, evento

Ejemplos

Request
{
  "file": "archivo.pdf",
  "tipo": "RESGUARDO"
}
Response
{

  "ruta": "uploads/..."
}
