checklist_items

Ítems de checklist.

Índice Inicio Swagger

Descripción

Detalle por checklist, puede asociarse a tipo de equipo.

Relación visual rápida

Campos clave

Obligatorios
  • checklistId
  • descripcion
Opcionales
  • tipoEquipoId
  • cantidad
  • obligatorio

Rutas y permisos

RutaRolNotas
GET /checklist-itemsAutenticadoFiltro: ?checklistId.
GET /checklist-items/:idAutenticadoPor id.
POST /checklist-itemsADMIN / GERENTECrea item.
PATCH /checklist-items/:idADMIN / GERENTEActualiza.
DELETE /checklist-items/:idADMIN / GERENTEElimina.

Controller (métodos)

crear
  • Zod create.
  • Default cantidad=1, obligatorio=true.
actualizar
  • Permite desconectar tipoEquipo.
  • Audit ENTITY_UPDATE.

Modelo / Validaciones

  • checklistItemCreateSchema: checklistId + descripcion.
  • checklistItemUpdateSchema: campos opcionales + nullables.

Ejemplos (requests / responses)

POST /checklist-items
{
  "checklistId": 7,
  "descripcion": "Laptop asignada",
  "cantidad": 1
}
201 OK
{
  "id": 30,
  "checklistId": 7,
  "obligatorio": true
}

Errores comunes

HTTPCausaSolución
