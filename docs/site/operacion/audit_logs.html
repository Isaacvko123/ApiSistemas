audit_logs

Bitácora de acciones críticas.

Índice Inicio Swagger

Descripción

Permite consultar eventos por actor, acción, objetivo y rango de fechas.

Relación visual rápida

Campos clave

Claves
  • action
  • actorId
  • targetType
  • targetId
  • createdAt
Opcionales
  • metadata
  • ip
  • userAgent

Rutas y permisos

RutaRolNotas
GET /audit-logsADMIN / GERENTEFiltros por acción, actor, fechas.
GET /audit-logs/:idADMIN / GERENTEObtiene por id.

Controller (métodos)

listar
  • Valida filtros con Zod.
  • Limita resultados (default 200, max 500).
obtener
  • Valida id.

Modelo / Validaciones

  • auditLogFilterSchema: action, actorId, targetType, targetId, dateFrom/dateTo.
