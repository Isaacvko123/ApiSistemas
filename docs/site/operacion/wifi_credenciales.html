wifi_credenciales

Credenciales WiFi cifradas.

Índice Inicio Swagger

Descripción

Almacena SSID/usuario/password cifrado. Sección de secretos restringida.

Relación visual rápida

Campos clave

Obligatorios
  • ssid
  • usuario
  • password (cifrado)
Opcionales
  • empleadoId
  • localidadId
  • notas
  • vigente

Rutas y permisos

RutaRolNotas
GET /wifi-credencialesAutenticadoFiltros: vigente, empleadoId, localidadId, ssid.
GET /wifi-credenciales/:idAutenticadoSin password.
GET /wifi-credenciales/:id/secretADMIN / GERENTEDevuelve password.
POST /wifi-credencialesADMIN / GERENTECrea.
PATCH /wifi-credenciales/:idADMIN / GERENTEActualiza.
DELETE /wifi-credenciales/:idADMIN / GERENTEvigente=false.

Controller (métodos)

obtenerSecreto
  • Audit ENTITY_READ_SECRET.
crear
  • Cifra password.
  • Audit ENTITY_CREATE.
actualizar
  • Permite desconectar relaciones.
  • Audit ENTITY_UPDATE.

Modelo / Validaciones

  • wifiCredencialCreateSchema: ssid/usuario/password.
  • wifiCredencialUpdateSchema: opcional + nullable.
  • toWifiCredencialPublic: oculta ciphertext.

Ejemplos (requests / responses)

POST /wifi-credenciales
{
  "ssid": "Oficina",
  "usuario": "admin",
  "password": "SecretWiFi"
}
200 OK
{
  "id": 3,
  "ssid": "Oficina",
  "usuario": "admin",
  "vigente": true
}
