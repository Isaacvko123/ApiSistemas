Resguardo equipos

Detalle de entrega

Índice Inicio Swagger

Descripción

Detalle por equipo en un resguardo (entrega y devolución).

Rutas y permisos

MétodoRutaDescripciónQueryAuthRoles
POST/resguardo-equiposAgrega equipo a resguardo-SíADMIN, GERENTE
GET/resguardo-equiposLista detallespage:number, pageSize:number, resguardoId:number, equipoId:numberSí-
GET/resguardo-equipos/{id}Obtiene resguardo equipos-Sí-
PATCH/resguardo-equipos/{id}Actualiza resguardo equipos-SíADMIN, GERENTE
DELETE/resguardo-equipos/{id}Elimina resguardo equipos-SíADMIN, GERENTE

Controller (métodos)

  • list
  • create

Modelo (campos clave)

  • id
  • resguardoId
  • equipoId
  • fechaEntrega
  • fechaDevolucion

Validaciones

  • resguardoId y equipoId válidos
  • Paginación: page, pageSize
  • Filtros: resguardoId, equipoId

Ejemplos

Request
{
  "resguardoId": 1,
  "equipoId": 2
}
Response
{
  "id": 1
}

Errores comunes

  • 404 resguardo/equipo no encontrado

Checklist operativo

  • Documento de entrega/devolución
