Usuarios

Gestión de cuentas y roles

Índice Inicio Swagger

Descripción

Crea, lista y administra usuarios de la plataforma. Integra RBAC y hash Argon2.

Rutas y permisos

MétodoRutaDescripciónQueryAuthRoles
POST/usuariosCrea usuario-No-
GET/usuariosLista usuariospage:number, pageSize:number, activo:boolean, rol:stringSíADMIN, GERENTE
GET/usuarios/{id}Obtiene usuarios-Sí-
PATCH/usuarios/{id}Actualiza usuarios-SíADMIN, GERENTE
DELETE/usuarios/{id}Elimina usuarios-SíADMIN, GERENTE

Controller (métodos)

  • list
  • create

Modelo (campos clave)

  • id
  • nombre
  • email
  • contrasenaHash
  • rol
  • activo
  • createdAt

Validaciones

  • email único
  • password >= 10
  • rol válido
  • Paginación: page, pageSize
  • Filtros: activo, rol

Ejemplos

Request
{
  "nombre": "Ana Pérez",
  "email": "ana@empresa.com",
  "contrasena": "PasswordSegura123",
  "rol": "ADMIN"
}
Response
{
  "id": 1,
  "nombre": "Ana Pérez",
  "email": "ana@empresa.com",
  "rol": "ADMIN"
}

Errores comunes

  • 400 payload inválido
  • 401 sin auth
  • 409 email duplicado

Checklist operativo

  • Hash Argon2
  • Auditoría de creación
  • No exponer contrasena
