Descripción

Puestos ligados a un área. Usados en empleados y checklist.

Rutas y permisos

MétodoRutaDescripciónAuthRoles
POST/puestosCrea puestoSíADMIN, GERENTE
GET/puestosLista puestosSí-
GET/puestos/{id}Obtiene puestosSí-
PATCH/puestos/{id}Actualiza puestosSíADMIN, GERENTE
DELETE/puestos/{id}Elimina puestosSíADMIN, GERENTE

Controller (métodos)

  • list
  • create

Modelo (campos clave)

  • nombre
  • areaId

Validaciones

  • areaId existente

Ejemplos

Request
{
  "nombre": "Soporte",
  "areaId": 1
}
Response
{
  "id": 1,
  "nombre": "Soporte",
  "areaId": 1
}

Errores comunes

  • 404 área no encontrada
  • 409 duplicado
