Tipos de equipo

Catálogo de dispositivos

Índice Inicio Swagger

Descripción

Catálogo maestro usado por equipos y checklist_items.

Rutas y permisos

MétodoRutaDescripciónQueryAuthRoles
POST/tipos-equipoCrea tipo de equipo-SíADMIN, GERENTE
GET/tipos-equipoLista tipos de equipopage:number, pageSize:number, nombre:string, activo:booleanSí-
GET/tipos-equipo/{id}Obtiene un tipo de equipo-Sí-
PATCH/tipos-equipo/{id}Actualiza tipo de equipo-SíADMIN, GERENTE
DELETE/tipos-equipo/{id}Elimina tipo de equipo-SíADMIN, GERENTE

Controller (métodos)

  • list
  • get
  • create
  • update
  • delete

Modelo (campos clave)

  • id
  • nombre
  • descripcion
  • activo

Validaciones

  • nombre único
  • descripcion opcional
  • activo boolean
  • Paginación: page, pageSize
  • Filtros: nombre, activo
  • No se puede eliminar si hay equipos/checklist_items relacionados.

Ejemplos

Request
{
  "nombre": "Laptop",
  "descripcion": "Equipo portátil",
  "activo": true
}
Response
{
  "id": 1,
  "nombre": "Laptop",
  "activo": true
}

Errores comunes

  • 400 payload inválido
  • 409 duplicado
  • 404 no encontrado
  • 409 relación inválida

Checklist operativo

  • Auditoría de cambios
  • No borrar si hay equipos ligados
